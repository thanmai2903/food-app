# 🍛 South Indian Flavors – Food Ordering Website

A modern, fully responsive **food ordering web application** built using **React.js** and **Tailwind CSS**, inspired by real-world platforms like **Swiggy** and **Zomato**.

This project provides a **complete food ordering experience** including menu browsing, cart management, checkout, payment simulation, live tracking, and order history.

---

## ✨ Features

### 🍽️ Food & Menu
- 🔍 Smart search with live filtering  
- 🎤 Voice search using Web Speech API  
- 🍴 Category-based filtering (Tiffins, Meals, Snacks, Sweets)  
- 🟢 Veg / 🔴 Non-Veg badges  
- ⭐ Bestseller tags  
- 🧾 Dynamic food data with images  

---

### 🛒 Cart & Ordering
- 🛒 Add to cart with quantity controls (+ / −)  
- 🧺 Floating cart sidebar (Swiggy-style)  
- 🔢 Real-time cart total calculation  
- 🔔 “Item added” toast notifications  
- 🚀 Quick “Order Now” (instant checkout)  

---

### 💳 Checkout & Payment
- 💳 Payment modal (UPI / Cash on Delivery)  
- ⏳ Payment loading simulation  
- ✅ Order confirmation flow  
- 🧹 Auto cart clear after payment  

---

### 🚚 Order Flow (Real App Experience)
- 📦 Order tracking page (step-by-step progress)  
- 🛵 Delivery animation (moving rider simulation)  
- 📍 Map integration (Google Maps support)  
- 🔔 Live notifications (order updates)  
- 🎉 Order success & completion pages  

---

### 📦 Orders & History
- 🧾 Save orders using localStorage  
- 📋 View past orders page  
- ⭐ Rating system after delivery  

---

### 👤 User Features
- 🔐 Login / Signup pages  
- 📱 Fully responsive (mobile, tablet, desktop)  
- 💬 WhatsApp quick contact button  

---

### 🎨 UI / UX
- ✨ Smooth animations (Framer Motion)  
- 🎟️ Coupon popup (discount UI)  
- ⭐ Customer reviews carousel  
- 🎯 Professional Swiggy-like UI  
- 📌 Sticky buttons & floating cart
